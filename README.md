# Intern Tracker App - MERN Stack

Welcome to the **Intern Tracker** project!

This is a take-home assessment where you will build a full-stack application using the **MERN stack** (MongoDB, Express, React, Node). The goal is to create a basic CRUD app where you can manage and track intern data.

### Functional Requirements

- **Data Model**: Intern collection with fields like name, email, role, status, score, createdAt, updatedAt.
- **Backend**: Build RESTful API with Node and Express to manage intern data.
- **Frontend**: Create a UI in React to view, add, edit, and delete interns with the ability to search, filter, and paginate the data.

### Tasks

1. **Backend (Node + Express)**
   - Implement API to create, list, get, update, and delete interns.
   - Handle input validation, error handling, and pagination.
   - Use MongoDB with Mongoose for storing data.

2. **Frontend (React)**
   - Build a UI to list interns with the ability to search, filter, and paginate.
   - Implement forms to add and edit interns with validations.
   - Handle delete functionality with confirmation dialog.
   - Display loading indicators and API error messages.
